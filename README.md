In de arbeidspsychologie is het JD-R model de gouden standaard voor het begrijpen van welzijn op het werk:

Job Demands (Eisen): Factoren die energie kosten (bijv. tijdsdruk, emotionele belasting, complexe omgeving).

Job Resources (Hulpbronnen): Factoren die energie geven en beschermen tegen burn-out (bijv. autonomie, coaching, flexibiliteit).
